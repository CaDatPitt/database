About CaD@Pitt

CaD@Pitt is the parent project of CaDatPittDB. CaD@Pitt (short for Collections as Data at Pitt) project is based in the University of Pittsburgh Library System and aims to increase the visibility and discoverability of library collections, make library collections data accessible for computational use, teach critical and computationally minded data practices, and enable scholars to extend/enrich collections data with research-driven data.
