About

CaD@PittDB

CadatPittDB is a subproject of CaD@Pitt, with the main purpose of giving researchers access to collections and allowing them to download collections data more independently, without mediation through library staff. This project upgrades the existing workflow for data extraction and transformation by simplifying the overall process and utilizing APIs to access data more directly, instead of using interfaces for repositories.

CaD@Pitt

CaD@Pitt is the parent project of CaDatPittDB. CaD@Pitt (short for Collections as Data at Pitt) project is based in the University of Pittsburgh Library System and aims to increase the visibility and discoverability of library collections, make library collections data accessible for computational use, teach critical and computationally minded data practices, and enable scholars to extend/enrich collections data with research-driven data..
